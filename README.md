# My Portfolio


## For Live Demo 

https://shubhamkhirwadkar.netlify.app/

## Installation

### Git repo clone

    git clone https://github.com/khirwadkarshubham25/my_portfolio.git

### Install Dependency

    npm install

### Start Server

    npm start

### Build Application

    npm run build
